WebGL Streamline Visualizer Examples

Explore the following examples to see how the WebGL Streamline Visualizer can be used:

  • Vortex Demo - A standalone example that generates velocity data using TypeScript functions.
  • Vortex with Sprites Demo - Similar to the vortex demo but with sprite-based visualization.
  • MapLibre Demo - Integrates the visualizer with a MapLibre map and a WMS service.
